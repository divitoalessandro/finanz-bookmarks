# finance-bookmarks

Kuratierte Ressourcen, Tools und Begriffe für persönliche Finanzen in der Schweiz.

## Tech Stack

- Plain HTML (four pages, no build process)
- [Pico.css](https://picocss.com/) via CDN (classless)
- Hosted on GitHub Pages

## Getting Started

```bash
git clone <repo-url>
cd finance-bookmarks
open index.html
```

That's it. Edit any `.html` file, reload the browser. No dev server, no build step, no dependencies to install.

## Pages

- `index.html` — Start / Navigation
- `resources.html` — Blogs, Podcasts, YouTube
- `tools.html` — Rechner, Vergleichsportale, Broker, Apps
- `glossary.html` — A–Z Glossar

See [PLAN.md](PLAN.md) for content decisions and open items.
